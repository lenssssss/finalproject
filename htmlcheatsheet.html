HTML CHEAT SHEET

The HTML of a website serves as the place where we code our TEXTS or the CONTENT that we want to show to our viewers. In HTML we will identify parts of our code with CLASS, ID, and ELEMENTS. Aside from this the HTML is where we place divisions with the div tag or containers in order to efficiently utilize our CSS. This is also where we place the images that we want to use.

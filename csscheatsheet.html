CSS CHEAT SHEET

CSS or Cascading Style Sheets refer to the way that the code is being read by the computer. It would read the code from top to bottom, this replaces any selector that coincides with the code before. This language is extremely important because this would give life to the website and make it nice and pretty to look at by the viewers.

  • CSS-GRID AND FLEXBOX IS A MUST TO LEARN FOR CSS. This assists us in the placement of texts, body, divs, images, and others in the website.
  • SELECTORS needs time to get used to
  • NEEDS this in order to make responsive UI

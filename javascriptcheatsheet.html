JAVASCRIPT CHEAT SHEET

Javascript is the hardest and most important part of the website. This ensures that the website is responsive. These are where the actions are done through the use of functions. Examples of these actions are those events that occur due to clicking, hovering, and other ways to interact with a website. These directly interact with the DOM of a website There are plenty of javascript tags that need to be learned in order to make 100% use of this. SYNTAX for this language is hard to learn and takes a lot of time to get used to. The code for javascript must be as effective as possible, meaning that it shouldnt be repetitive.

JAVASCRIPT SYNTAX

Syntax Use Delete
let Declaring Variables
console.log Console Output
if-else conditional statement let num = 10; if (num > 0) { console.log("Positive"); } else { console.log("Non-positive"); }
Ternary Operator (Conditional Expression): let result = (num > 0) ? "Positive" : "Non-positive"; console.log(result);
Function Declaration function greet(name) { console.log("Hello, " + name + "!"); }
Function Call greet("Alice");
Array Declaration let numbers = [1, 2, 3, 4, 5]
Array Access console.log(numbers[0]); // Outputs: 1
Array Length console.log(numbers.length); // Outputs: 5
Loop (for) for (let i = 0; i < numbers.length; i++) { console.log(numbers[i]); }
Loop (while) let i = 0; while (i < numbers.length) { console.log(numbers[i]); i++; }
Loop (for...of) for (let number of numbers) { console.log(number); }
Loop (forEach) numbers.forEach(function(number) { console.log(number); });
Array Push numbers.push(6)
Array Pop let lastNumber = numbers.pop()
